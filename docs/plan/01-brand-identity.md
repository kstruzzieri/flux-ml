# Flux Brand Identity

## Name
**Flux**

## Taglines
- **Primary descriptor:** "The ML development environment"
- **Marketing hook:** "Write code. Watch it learn."

These can be used together or separately:
- Repo/app header: Both together
- Casual mention: Just "Flux"
- Landing page: Full treatment with both

## Why "Flux"
- Represents change, flow, continuous movement
- Training is literally gradient *flow*
- Experiments are in constant *flux*
- Short, memorable, ownable

## Positioning
Flux is NOT:
- A "better VS Code"
- A generic ML experiment tracker
- A clone of MLflow/W&B

Flux IS:
- An integrated ML development environment
- Where code and training live together
- A tool that understands what you're doing, not just displays text
- Purpose-built for reward model development workflows

## Key Differentiators

| vs MLflow/W&B | Flux Advantage |
|---------------|----------------|
| Server required | SQLite, works offline |
| Separate from code | Inline with editor |
| Generic ML | Reward model-specific |
| Manual analysis | Auto-detection of pathologies |
| 100s MB | ~15MB binary |
